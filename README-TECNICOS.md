# ION - Para Técnicos

## ¿Qué es ION?

ION es un agente de IA que te ayuda en tu trabajo diario en IONET:
- Protocolos y procedimientos de soporte
- Información sobre clientes, redes, inventario
- Redacción de tickets y documentación
- Consultas técnicas generales

ION coordina 7 agentes especializados para responder tus dudas.

---

## Cómo Usar ION

### Por Email (Principal)

Envía un email a: **soporte@ionet.cl**

Ejemplos:
```
Asunto: Protocolo reset router
Mensaje: Necesito el procedimiento para reiniciar un router Huawei

Asunto: Topología red cliente X
Mensaje: Cuál es la IP del servidor VPN del cliente X?

Asunto: Nuevo usuario
Mensaje: Cómo agregar un usuario nuevo al cliente Y?
```

ION responderá en menos de 30 segundos.

### Por Web UI (Opcional)

Si tienes acceso admin:
1. Entra a: http://servidor-internal:8080
2. Chatea directamente con ION

### Por Teams (Solo recibe notificaciones)

ION te enviará alertas importantes al canal de Teams:
- Incidentes críticos
- Escalamientos a humanos
- Resúmenes de actividad

---

## Agentes Especializados

ION deriva automáticamente al agente correcto:

| Pregunta sobre... | ION deriva a |
|------------------|--------------|
| Procedimientos, políticas | agente-protocolos |
| Redes, IPs, servidores | agente-redes |
| Clientes, contratos | agente-clientes |
| Tickets, SLAs | agente-servicios |
| Equipos, licencias | agente-inventario |
| Seguridad, vulnerabilidades | agente-seguridad |
| Documentos, backups | agente-datos |
| Contactos, FAQ | agente-base |

## Ejemplos de Consultas

**Protocolos:**
- "Protocolo para reinstalar Office"
- "Cómo hacer onboarding de cliente nuevo"
- "Procedimiento de escalado L1 a L2"

**Redes:**
- "IP del servidor de respaldos"
- "Configurar VPN para cliente X"
- "Topología de red del cliente Y"

**Seguridad:**
- "Tengo alerta de ransomware, qué hago?"
- "Cómo reportar incidente de seguridad"
- "Procedimiento para usuario bloqueado"

**Clientes:**
- "Contratos del cliente X"
- "Usuarios del cliente Y"
- "SLAs del cliente Z"

## Lo que ION SÍ Puede Hacer

✅ Responder preguntas sobre documentación
✅ Ayudar a redactar tickets
✅ Buscar procedimientos operativos
✅ Derivar a humano cuando es necesario
✅ Enviar alertas a Teams
✅ Proporcionar contexto de clientes y redes

## Lo que ION NO Puede Hacer

❌ Acceder a sistemas en tiempo real (RDP, SSH)
❌ Hacer cambios en configuración
❌ Ejecutar comandos en servidores
❌ Acceder a información no documentada

Para estas acciones, ION te indicará cómo proceder o escalará a humano.

---

## Tiempos de Respuesta

- Consultas simples: < 5 segundos
- Consultas complejas: < 30 segundos
- Derivaciones: < 10 segundos adicionales

---

## Soporte

Si ION no funciona:
1. Envía email a soporte-it@ionet.cl
2. Reporta el incidente en el sistema de tickets
3. Describe: qué intentaste, qué respuesta esperabas

---

## Tips

✅ Sé específico en tu pregunta
✅ Menciona el nombre del cliente si aplica
✅ Incluye contexto relevante (modelo de equipo, error, etc.)

❌ No incluyas información sensible (passwords, datos personales)
❌ No esperes que ION acceda a sistemas externos
❌ No uses ION para acciones que requieren aprobación
